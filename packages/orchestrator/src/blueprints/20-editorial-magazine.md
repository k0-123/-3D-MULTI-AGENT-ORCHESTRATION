# Editorial Serif Magazine Blueprint

## Aesthetic Overview
- **Theme**: Classic Editorial / Avant-Garde Publication
- **Color Palette**: `#ECE8E1` newsprint background, `#111111` deep black text, `#D9383A` crimson highlight.
- **Typography**: Playfair Display (Serif Headers) & EB Garamond (Body).

## Key Components
1. **Asymmetric Editorial Layout**: Multi-column text layouts with oversized drop caps and dramatic whitespace.
2. **Cinematic Pull Quotes**: Large italicized callouts with custom CSS border accents and subtle background grain.
3. **Parallax Image Reveal**: High-resolution photography with CSS mask overlays that reveal on scroll.
