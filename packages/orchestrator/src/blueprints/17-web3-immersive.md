# Web3 Immersive Protocol Blueprint

## Aesthetic Overview
- **Theme**: Dark cyberpunk / Web3 native
- **Color Palette**: `#0A0B10` background, `#7B61FF` primary glow, `#00F0FF` cyan accents, `#FFFFFF` text.
- **Typography**: Space Grotesk (Headers) & Inter (Body).

## Key Components
1. **Neon Vector Grid**: Background animated SVG grid with glowing nodes.
2. **Glassmorphic Protocol Dashboard**: Floating translucent panels (`backdrop-filter: blur(20px)`) displaying live gas fees and TVL metrics.
3. **Interactive Terminal Widget**: Simulated command-line interface for staking and swapping tokens.
