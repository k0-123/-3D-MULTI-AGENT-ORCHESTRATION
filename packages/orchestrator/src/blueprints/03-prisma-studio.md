# Blueprint 03: Prisma Creative Studio

## Visual Goal
Dark, moody, and cinematic with a warm cream color palette. Uses fractal noise overlays and WordsPullUp text animations.

## Tech Stack
- React + Vite + TypeScript
- Tailwind CSS
- framer-motion
- lucide-react

## Design Tokens
- Background: #000000
- Primary: #DEDBC8 (Warm Cream)
- Font Global: 'Almarai'
- Font Accent: 'Instrument Serif' (Italic)

## Components
1. **Hero**: Inset container with rounded-2xl edges and background video.
2. **WordsPullUp**: Word-by-word slide-up animation.
3. **About Card**: Scroll-linked character opacity reveal.
