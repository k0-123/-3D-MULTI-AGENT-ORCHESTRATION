# Blueprint 04: Toonhub 3D Carousel

## Visual Goal
Character-figurine carousel with role-based transitions and a giant "3D SHAPE" ghost text.

## Tech Stack
- React + Vite
- Lucide React
- Grain Overlay (SVG)

## Design Tokens
- Colors: Dynamic based on active figurine (Orange, Green, Pink, Blue)
- Font Display: 'Anton'
- Font Body: 'Inter'

## Components
1. **Carousel Logic**: Role-based (left, center, right, back) positioning.
2. **Grain Overlay**: Subtle SVG noise texture.
3. **Ghost Text**: Massive background typography.
