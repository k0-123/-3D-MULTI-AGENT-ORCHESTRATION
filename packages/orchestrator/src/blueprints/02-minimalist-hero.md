# Blueprint 02: Velorah Minimalist Hero

## Visual Goal
Create a single-page hero section with a fullscreen looping background video, glassmorphic navigation, and cinematic typography.

## Tech Stack
- React + Vite
- Tailwind CSS
- shadcn/ui (patterns)

## Design Tokens
- Background: 201 100% 13% (Deep Navy)
- Foreground: White
- Font Display: 'Instrument Serif'
- Font Body: 'Inter'

## Components
1. **Video Background**: Fullscreen inset-0 object-cover.
2. **Liquid Glass**: .liquid-glass class with background-blend-mode: luminosity.
3. **Animations**: @keyframes fade-rise.
